# MoboMojo - PC Build Planner

A modern PC build planning application with compatibility checking, price tracking, and cross-device authentication.

Live Demo: https://agsiangio.github.io/HCI-2-MoboMojo-Project/

## Features

- Cross-device authentication via Supabase
- Smart compatibility checker with automatic suggestions
- Build saver with database persistence
- Wishlist management
- Price tracking for PC components
- Community reviews
- Modern, tech-themed UI

## Setup

1. Open `index.html` in a web browser
2. Sign up for an account
3. Start building your PC!

## How to Use

### Compatibility Checker
Enter any PC component (CPU or GPU) to see compatible parts automatically:
- RTX 4070, RTX 4080
- Ryzen 5 7600X, Ryzen 7 7800X3D
- Intel i5-13600K, Intel i7-13700K

### Build Planner
1. Go to the Builder page
2. Enter all your components (CPU, GPU, Motherboard, RAM, PSU, Storage, Cooler, Case)
3. Save your build with a custom name
4. Your builds are saved to the database and accessible from any device

### Cross-Device Access
- Sign up on one computer
- Log in on another computer with the same credentials
- All your builds and wishlist items are automatically synced

## Technology Stack

- HTML, CSS, JavaScript
- Supabase for authentication and database
- Modern responsive design with PC building theme
